## Management Command

The jobs were created using a custom Django management command:

```bash
python manage.py create_job_descriptions
```

This command:
- Creates necessary departments (Technology, Marketing, QA, Design)
- Creates or retrieves designations
- Creates 13 comprehensive job descriptions
- Sets all jobs to "Active" status
- Sets application deadline to 45 days from creation
